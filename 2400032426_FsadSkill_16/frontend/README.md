# Skill 16 Frontend

React frontend for the Swagger-documented Student CRUD application.

## Features
- Add, view, update, and delete students
- Works with the Skill 16 Spring Boot backend
- Includes a quick link to Swagger UI

## Run

```bash
cd frontend
npm install
npm run dev
```

Backend Swagger URL:

```text
http://localhost:8080/swagger-ui.html
```
